## Quality Gate

Every content piece must pass `docs/GROWTH_CONTENT_EXECUTION_CHECKLIST.md` before publish:
- Direct answer intro (40-60 words)
- Query-style headings
- Claim -> mechanism -> evidence structure
- Internal links to tools + pricing
- FAQ section (3-5 questions per page)
- Canonical terms only (`tokens`, `AISkillScore`)

## Measurement

Track weekly via PostHog + Search Console:
- Organic sessions per page
- Content-to-tool CTR (target: 6%+)
- FAQ snippet capture rate
- AI citation visibility for tracked queries
